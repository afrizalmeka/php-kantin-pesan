# KantinKu — Pemesanan Kantin Kampus

Aplikasi pemesanan kantin kampus berbasis PHP dan SQLite untuk keperluan pembelajaran pengujian perangkat lunak.

## Fitur
- Tampilan menu kantin dengan filter kategori
- Login & registrasi pelanggan
- Pembuatan pesanan (multi-item)
- Riwayat pesanan pelanggan
- Kelola menu & status pesanan (admin)

## Cara Menjalankan

1. **Clone atau unduh** folder `php-kantin-pesan`.
2. **Jalankan server PHP:**
   ```bash
   cd php-kantin-pesan
   php -S localhost:8004
   ```
3. **Buka browser** dan akses `http://localhost:8004`
